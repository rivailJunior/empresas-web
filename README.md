## Getting Started

Download or clone the project
    
    git clone https://github.com/rivailJunior/empresas-web.git

Dev Mode
```bash
npm install
or
yarn install

npm run dev
# or
yarn run dev
```
Prod Mode

      npm install
      or
      yarn install
     npm run build
     # and
     npm run start

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
